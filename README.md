# cairo-cr

Cairo bindinds for Crystal language.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  cairo-cr:
    github: TamasSzekeres/cairo-cr
```
Then run in terminal:
```bash
crystal deps
```

## Usage

```crystal
require "./cairo-cr/*"

module YourModule
  include CairoCr # For simpler use
end
```
